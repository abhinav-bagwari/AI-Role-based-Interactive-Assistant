# Sample Workflow Execution

## Requirement

Build a Director-led team that turns a product idea into code, documentation, tests, and deployment guidance.

## Selected Team

`team-b` (Full BMAD Product Squad)

## Example Protocol Flow

1. Director -> pm, analyst, architect: `TASK_ASSIGNMENT` (`discovery`)
2. pm/analyst/architect -> Director: `INTERMEDIATE_OUTPUT`
3. Director -> sm: `REVIEW_REQUEST`
4. sm -> Director: `REVIEW_FEEDBACK`
5. Director -> ux-designer, tech-writer: `TASK_ASSIGNMENT` (`design-and-docs`)
6. Director -> dev, qa: `TASK_ASSIGNMENT` (`build-and-verify`)
7. Director -> user: `FINAL_DELIVERY`

## Output Package

- PRD artifacts
- Architecture/design artifacts
- Documentation artifacts
- Code/API/deployment artifacts
- Test artifacts
- Full protocol trace
